# Recipe Generator
### Simple vue application that retrieves information from the MealDb API and renders a users chosen recipe in a simple layout

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
